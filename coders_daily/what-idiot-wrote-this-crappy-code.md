# 程序员的日常：哪个蠢蛋写的烂代码？

关键词：程序员的日常, 程序员漫画, 程序员的冷笑话, 代码维护, 好代码 烂代码

编码其实不是件容易的事情，编写易维护的代码更是难上加难。很多程序员在代码易维护性方面可能不太在意，毕竟完成眼前的工作才是最紧要的，至于之后代码由谁去维护这种问题，也不会去多想，最多在心里默默同情以后的代码维护者。不过看完这期《程序员的日常》漫画，你可能就不会这么想了。
